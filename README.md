# E-Commerce Gamifié (React + Express + MongoDB)

Projet minimal complet: frontend React (Vite) + backend Express + MongoDB (mongoose).

Instructions rapides:
- Backend: `cd backend` -> `npm install` -> `npm run dev` (requires Node.js)
- Frontend: `cd frontend` -> `npm install` -> `npm run dev`

Fichier de configuration example .env (backend/.env):
# 1. Backend
cd backend
npm install
# Créer .env avec MongoDB URI
npm run dev

# 2. Frontend
cd frontend  
npm install
npm run dev

# 3. Initialiser DB
cd backend
node scripts/seed.js
